Presentacion de Ruby

Ruby es un lenguaje de programación interpretado, orientado a objetos y de código abierto, creado en 1995 por Yukihiro "Matz" Matsumoto. Su diseño prioriza la simplicidad y la productividad, con una sintaxis clara y fácil de leer. Es un lenguaje multi-paradigma, ya que admite programación orientada a objetos, funcional y estructurada. Además, cuenta con un recolector de basura automático y no requiere compilación, lo que facilita el desarrollo rápido. Ruby es ampliamente utilizado en el desarrollo web gracias a Ruby on Rails, un framework que permite crear aplicaciones de manera eficiente. Su comunidad es activa y su versatilidad lo hace ideal para proyectos modernos.

Declaraciones de variables

En Ruby, declarar variables es muy sencillo, ya que no necesitas especificar el tipo de dato (es un lenguaje de tipado dinámico). Solo asignas un valor y Ruby lo interpreta automáticamente. 


            # Variable de tipo cadena (String)
            nombre = "Juan"

            # Variable de tipo entero (Integer)
            edad = 25

            # Variable de tipo flotante (Float)
            altura = 1.75

            # Variable booleana (Boolean)
            es_programador = true

            # Variable de tipo array (lista)
            colores = ["rojo", "azul", "verde"]

            # Variable de tipo hash (diccionario)
            persona = { nombre: "Ana", edad: 30, ciudad: "Madrid" }

            # Imprimir variables
            puts "Nombre: #{nombre}, Edad: #{edad}, Altura: #{altura}m"
            puts "¿Es programador? #{es_programador}"
            puts "Colores favoritos: #{colores.join(", ")}"
            puts "Datos de la persona: #{persona}"
